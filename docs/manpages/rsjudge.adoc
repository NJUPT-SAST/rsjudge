== Name

rsjudge - an online judge sandbox server in Rust

== Usage

*rsjudge* [_OPTIONS_]...

== Options

*-c, --config-dir* <DIR>::
    Specify the configuration directory [default: ./config]
*-h, --help*::
    Print help
*-V, --version*::
    Print version
